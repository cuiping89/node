
# 格式管理

## 1. 颜色系统
- 主色：`#10b981`（绿色）用于状态和成功
- 辅色：`#3b82f6`（蓝色）用于链接和详情
- 警告：`#f59e0b`（橘黄）用于图表和警告状态
- 背景：`#f5f5f5`（浅灰）、 `#f9fafb`（更浅灰）
- 文字：`#1f2937`（深灰）、`#4b5563`（中灰）、`#6b7280`（浅灰）

## 2.卡片背景
- 一级卡片：白色
- 二级卡片： `#f9fafb`（更浅灰）
- 三级卡片：`#f5f5f5`（浅灰）
